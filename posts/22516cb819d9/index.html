腾讯，请停止你的无端污名化！

2023年07月24日
杂文笔札
留言

上午分享我的生活琐事《水龙头维修记》时，微信竟然给该篇文章标记为包含色情内容，也不知道腾讯你是如何识别出这篇文章包含lewd内容的？虽然微信是你腾讯的地盘，但也不能做事太腾讯。 腾讯，请停止你的无端污名化！

《计算机世界》于2010年7月刊发一篇封面头条文章，《“狗日的”腾讯》，文中把腾讯作为互联网公敌进行批判，将互联网商业竞争写成了不可调和的恩怨情仇，详细叙述了联众、奇虎、360等公司与腾讯之间的爱恨纠葛。

时至今日，《计算机世界》因新冠疫情等各种原因停工停业。不得不说，十多年前，就有人看出你的本质，做生意使招不寒碜，但无端污名化别人，就是你腾讯的不对了。更何况《水龙头维修记》是我个人的生活琐事记录，怎么就lewd了？还弄出个莫名的据用户投诉，全世界的用户能有这么无聊嘛？

附微信页面，无端污名化中英文版：

已停止访问该网页

据用户投诉及腾讯安全网址安全中心检测，该网页包含色情内容。为维护绿色上网环境，已停止访问。

This page cannot be accessed now

Based on user reports and Tencent’s web security checks, it has been determined that this page contains lewd content. Access has been blocked to maintain a safe online environment.

以下是来自“腾讯安全-网址安全中心”的检测信息，不知道腾讯的色情内容拦截标准仅仅是技术问题，还是有其它目的？腾讯是不是滥用市场支配地位不得而知。
